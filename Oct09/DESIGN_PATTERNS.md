# Introduction

Design Patterns is the most essential part of Software Engineering, as they provide the general repeatable solution to a commonly occurring problem in software design. They usually represent some of the best practices adopted by experienced object-oriented software developers.

## Creational Design Patterns

- Factory Method
- Abstract Factory Method
- Builder Method
- Prototype Method
- Singleton Method

## Structural Design Patterns

- Adapter Method
- Bridge Method
- Composite Method
- Decorator Method
- Facade Method
- Proxy Method
- FlyWeight Method

## Behavioral Design Patterns

- Chain of Responsibility Method
- Command Method
- Iterator Method
- Mediator Method
- Memento Method
- Observer Method
- State Method
- Strategy Method
- Template Method
- Visitor Method
